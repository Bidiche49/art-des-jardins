## Description

Plusieurs informations juridiques sont incorrectes ou placeholders sur le site vitrine :

1. **Footer** : affiche `SIRET 123 456 789 00012` au lieu du vrai SIRET `999 636 806 00013`
2. **Schema.org LocalBusinessSchema** : utilise un identifier SIRET placeholder
3. **Schema.org foundingDate** : indique `2024` alors que la creation est le `09/01/2026`
4. **Mentions legales** : SIRET affiche sans le NIC (manque le 00013)
5. **Manque** : numero TVA intracommunautaire `FR36999636806`
6. **Manque** : capital social `5 000 EUR`
