# BUG-008: Accents manquants dans tous les meta titles, descriptions et contenus

**Type:** Bug
**Statut:** Fait
**Priorite:** Critique
**Complexite:** S
**Tags:** seo, vitrine, i18n
**Date creation:** 2026-02-13

---

## Description

Tous les textes du site vitrine (meta titles, meta descriptions, headings, contenu) sont ecrits **sans accents** : "Amenagement" au lieu de "Amenagement", "Elagage" au lieu de "Elagage", "Creation" au lieu de "Creation", etc.

Cela impacte directement le SEO car les utilisateurs Google tapent avec accents ("paysagiste Angers elagage") et Google fait un matching exact sur les caracteres accentues. Le CTR dans les SERP est aussi impacte car les meta descriptions sans accents paraissent non professionnelles.

## Impact SEO

- **Matching requetes** : Google distingue "elagage" et "elagage" dans le ranking
- **CTR SERP** : Les descriptions sans accents reduisent le taux de clic de 10-15%
- **Credibilite** : Un site professionnel francais DOIT avoir les accents
