# BUG-021: Placeholders non remplis dans pages legales

**Type:** Bug
**Statut:** A faire
**Priorite:** Haute
**Complexite:** S
**Tags:** vitrine, legal, contenu
**Date creation:** 2026-02-27

---

## Description

Plusieurs pages legales contiennent des placeholders non remplaces par les vraies informations. Ces pages sont publiques et l'absence d'informations reelles peut poser un probleme legal (obligation d'affichage).

### Occurrences

1. **Mentions legales** (`apps/vitrine/src/app/mentions-legales/page.tsx`)
   - Lignes 55-58 : Hebergement = "Nom: A definir", "Adresse: A definir", "Contact: A definir"
   - Remplir avec : Cloudflare Pages (Cloudflare, Inc. - 101 Townsend St, San Francisco, CA 94107, USA)

2. **Politique de confidentialite** (`apps/vitrine/src/app/politique-confidentialite/page.tsx`)
   - Ligne 39 : "[Adresse complete], 49000 Angers" → Remplacer par "9 bis rue Rouget de l'Isle, 49130 Les Ponts-de-Ce"
   - Ligne 146 : "[Adresse], 49000 Angers" → Idem

3. **CGV** (`apps/vitrine/src/app/cgv/page.tsx`)
   - Ligne 141 : "[Coordonnees du mediateur]" → Remplir avec un mediateur de la consommation agree
   - Note : obligation legale depuis 2016 pour tout professionnel (art. L612-1 Code de la consommation)

## Criteres d'acceptation

- [ ] Hebergeur renseigne dans mentions legales (Cloudflare Pages)
- [ ] Adresse correcte (9 bis rue Rouget de l'Isle, 49130 Les Ponts-de-Ce) dans politique de confidentialite (2 endroits)
- [ ] Mediateur de la consommation renseigne dans CGV
- [ ] Aucun placeholder restant dans les 3 pages
- [ ] Build OK apres modifications
