# BUG-010: Google Search Console verification non activee

**Type:** Bug
**Statut:** A faire
**Priorite:** Critique
**Complexite:** XS
**Tags:** seo, vitrine, infra
**Date creation:** 2026-02-13

---

## Description

La verification Google Search Console est commentee dans `layout.tsx` :
```typescript
verification: {
    // google: 'votre-code-verification-google',
},
```

Sans Search Console, impossible de :
- Soumettre le sitemap a Google
- Monitorer l'indexation des pages
- Voir les requetes de recherche
- Detecter les erreurs d'exploration
- Mesurer les performances SEO

C'est la **premiere chose a faire** pour tout site web visant le SEO.

## Fichiers concernes

- `apps/vitrine/src/app/layout.tsx` (ligne 89)

## Approche

1. Aller sur https://search.google.com/search-console
2. Ajouter la propriete `art-et-jardin.fr`
3. Choisir la methode de verification "Balise HTML"
4. Copier le code de verification
5. Decommenter et remplir la valeur dans layout.tsx
6. Deployer
7. Valider la verification dans Search Console
8. Soumettre le sitemap : `https://art-et-jardin.fr/sitemap.xml`
