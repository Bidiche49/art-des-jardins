## Description

Les boutons CTA actuels (`btn-primary`) sont fonctionnels mais visuellement basiques : fond vert plat, pas de gradient, pas d'icone, pas de micro-animation au hover.

Pour un positionnement premium et une meilleure conversion :
- Gradient subtil (primary-600 â†’ primary-700)
- Icone fleche ou chevron a droite
- Hover : translation + ombre + scale subtile
- Active : enfoncement (scale 0.98)
- Focus : ring visible pour accessibilite
