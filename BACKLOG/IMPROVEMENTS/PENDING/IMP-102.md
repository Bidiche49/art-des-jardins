# IMP-102: Schema sameAs — ajouter URLs annuaires au fur et a mesure

**Type:** Improvement
**Statut:** A faire
**Priorite:** Haute
**Complexite:** XS
**Tags:** seo, schema, local
**Date creation:** 2026-02-28

---

## Description
Le schema LocalBusiness du site doit lister les profils annuaires dans le champ `sameAs` pour renforcer les signaux de citation NAP (Name, Address, Phone) aupres de Google. Chaque profil cree = une URL a ajouter dans le code.

**Prerequis** : les profils doivent d'abord etre crees sur chaque plateforme.

## Fichier concerne
- `apps/vitrine/src/components/seo/LocalBusinessSchema.tsx` (tableau `sameAs`, lignes ~71-77)

## Implementation
Quand chaque profil est cree, ajouter l'URL dans le tableau `sameAs` :
```ts
sameAs: [
  'https://www.instagram.com/artdesjardins_49',
  'https://www.societe.com/societe/art-des-jardins-999636806.html',
  // Ajouter au fur et a mesure :
  // 'https://www.google.com/maps/place/Art+des+Jardins/...',
  // 'https://www.pagesjaunes.fr/pros/...',
  // 'https://www.facebook.com/artdesjardins49',
  // 'https://www.houzz.fr/pro/artdesjardins49',
],
```

## Profils a creer (action paysagiste)

### 1. Google My Business — CRITIQUE
**Ce que ca rapporte :** C'est LE profil le plus important. Il fait apparaitre l'entreprise dans Google Maps et dans l'encart local (les 3 resultats avec carte qui s'affichent en haut de Google quand quelqu'un cherche "paysagiste Angers"). Sans ca, l'entreprise est quasi invisible en recherche locale. C'est aussi la que les clients laissent des avis Google — chaque avis 5 etoiles pousse le classement.
- Creer sur https://business.google.com
- Renseigner : nom, adresse 9B rue Rouget de l'Isle 49130 Les Ponts-de-Ce, tel, horaires, photos
- Verifier le profil (courrier ou telephone)

### 2. Pages Jaunes
**Ce que ca rapporte :** Pages Jaunes est le 1er annuaire francais. Google considere une fiche PJ comme un signal de confiance fort ("cette entreprise existe vraiment"). Beaucoup de particuliers 40+ cherchent encore sur PJ. La fiche basique est gratuite.
- Creer sur https://www.pagesjaunes.fr/inscription
- Renseigner les memes infos (NAP identique a Google)

### 3. Facebook (page entreprise)
**Ce que ca rapporte :** Une page Facebook pro permet de poster des photos avant/apres de chantiers, recevoir des avis, et etre trouve par les gens qui cherchent "paysagiste" sur Facebook. Meme sans publier souvent, la page existe et renforce la presence en ligne. Google indexe les pages Facebook.
- Creer une Page (pas un profil perso) sur https://www.facebook.com/pages/create
- Categorie : Paysagiste / Service d'amenagement paysager

### 4. Houzz.fr
**Ce que ca rapporte :** Houzz est LE site de reference pour les projets maison/jardin. Les particuliers qui preparent un amenagement y cherchent des pros. Un profil avec quelques photos de realisations peut generer des contacts qualifies de gens prets a investir. C'est aussi un backlink de qualite pour le SEO.
- Creer sur https://www.houzz.fr/professionals
- Ajouter des photos de realisations

### 5. Yelp (optionnel)
**Ce que ca rapporte :** Moins utilise en France que dans les pays anglophones, mais Yelp est quand meme indexe par Google. Ca fait un signal de citation NAP supplementaire a cout zero. A faire si le temps le permet.
- Creer sur https://biz.yelp.fr
