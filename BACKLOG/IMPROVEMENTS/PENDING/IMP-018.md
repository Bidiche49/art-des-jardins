# IMP-018: Decouper le composant Calendar monolithique

**Type:** Improvement
**Statut:** A faire
**Priorite:** Moyenne
**Complexite:** M
**Tags:** pwa, ui
**Date creation:** 2026-02-12

---

## Description

Le composant `Calendar.tsx` fait 760+ lignes avec logique drag & drop, filtres, export iCal, legende employes, toolbar, et rendu evenements. Difficile a maintenir et tester.

## Criteres d'acceptation

- [ ] Calendar decoupe en sous-composants : CalendarHeader, CalendarLegend, ExportModal
- [ ] Hook custom `useCalendarDragAndDrop` extrait
- [ ] Fichier principal < 200 lignes
- [ ] Fonctionnalite identique (pas de regression)

## Fichiers concernes

- `apps/pwa/src/pages/Calendar.tsx` - a decouper
- `apps/pwa/src/components/calendar/` - sous-composants a creer

## Analyse / Approche

```
Calendar.tsx (orchestrateur, <200 lignes)
├── components/calendar/CalendarHeader.tsx (toolbar + filtres)
├── components/calendar/CalendarLegend.tsx (employes + couleurs)
├── components/calendar/CalendarExportModal.tsx (modal iCal)
├── components/calendar/CalendarEvent.tsx (existant)
└── hooks/useCalendarDragAndDrop.ts (logique DnD)
```

## Tests de validation

- [ ] Calendrier fonctionne identiquement (drag, filtres, export)
- [ ] Calendar.tsx < 200 lignes
