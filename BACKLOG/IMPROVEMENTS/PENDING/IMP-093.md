## Description

Les chiffres cles (16+, 500+, 100%, 48h) sont affiches en texte basique dans le StatsCounter. Les sites premium utilisent :
- Polices display extra-large (text-5xl ou plus)
- Couleurs distinctives (or/secondary pour les chiffres)
- Compteurs animes plus sophistiques (easing, rebond)
- Separateur visuel entre les stats (barre verticale ou icone)
- Fond avec texture ou gradient subtil

## User Story
**En tant que** visiteur
**Je veux** etre impressionne par les chiffres cles
**Afin de** retenir les arguments de l'entreprise

## Criteres d'acceptation
- [ ] Chiffres en taille XL avec police serif (Cormorant)
- [ ] Couleur secondary-400 pour les chiffres
