## Description

Le site utilise `https://art-et-jardin.fr` comme domaine par defaut dans `layout.tsx`. Quand le vrai nom de domaine sera achete, il faudra le mettre a jour.

**Impact si oublie :** Tous les canonical URLs, OG URLs, sitemap, et schemas JSON-LD pointeront vers un domaine inexistant â†’ SEO casse.

## Fichier a modifier

- `apps/vitrine/src/app/layout.tsx` ligne 17 : `const siteUrl = 'https://art-et-jardin.fr';`

Changer en :
```typescript
const siteUrl = 'https://NOUVEAU-DOMAINE.fr';
```

## Verification apres changement

- Grep `art-et-jardin.fr` dans tout le projet pour verifier qu'il n'y a pas d'URLs hardcodees ailleurs
- Verifier les canonicals, OG, sitemap.xml, schemas JSON-LD
