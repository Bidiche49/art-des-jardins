## Description

La font Inter est chargee sans `display: 'swap'`, ce qui peut causer un Flash of Invisible Text (FOIT). Le texte est invisible pendant le chargement de la font.

## Criteres d'acceptation

- [ ] `display: 'swap'` ajoute a la config Inter
- [ ] Variable CSS `--font-inter` definie
- [ ] Pas de FOIT visible au chargement

## Fichiers concernes

- `apps/vitrine/src/app/layout.tsx:8`

## Analyse / Approche

```typescript
const inter = Inter({
  subsets: ['latin'],
  display: 'swap',
  variable: '--font-inter',
});
```

## Tests de validation

- [ ] Build OK
- [ ] Texte visible immediatement au chargement (pas de flash invisible)
- [ ] Lighthouse Performance score stable ou ameliore
