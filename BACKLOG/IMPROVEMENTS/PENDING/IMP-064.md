# IMP-064: Mode terrain complet (gants, soleil, contrastes)

**Type:** Improvement
**Statut:** A faire
**Priorite:** Critique
**Complexite:** M
**Tags:** accessibility, terrain, ux
**Date creation:** 2026-02-25

---

## Description

TerrainTheme n'affecte que ElevatedButton (pas FilledButton, OutlinedButton, TextButton, IconButton). Boutons 44dp vs 56-64dp requis avec gants de travail. Contrastes insuffisants au soleil (grey500 sur fond blanc = 5:1 limite). Espacement 8dp trop serre entre cibles tactiles.

## Findings associes

- TERRAIN-001: Contraste insuffisant mode clair pour soleil
- TERRAIN-003: Polices trop petites (bodySmall 12px, labelSmall 11px)
- TERRAIN-004: Boutons 44x48dp vs 56-64dp requis (CRITIQUE)
- TERRAIN-005: Espacement 8dp insuffisant entre cibles
- TERRAIN-014: TerrainTheme incomplet (que ElevatedButton)
- TERRAIN-015: Pas de feedback visuel activation mode terrain

## Criteres d'acceptation

- [ ] TerrainTheme applique a TOUS types boutons (Elevated, Filled, Outlined, Text)
- [ ] Taille minimale 56dp pour tous les boutons en mode terrain
- [ ] IconButton constraints minimum 48x48dp (56dp en terrain)
- [ ] Espacement minimum 12dp entre cibles tactiles
- [ ] Contrastes augmentes: grey500 â†’ grey700 pour textes corps
- [ ] Polices min 16px bodySmall, 15px labelSmall en terrain
- [ ] Chip/snackbar de confirmation a l'activation du mode terrain
