## Description
Google My Business est LE levier SEO local numero 1. 45% des clics locaux vont au "pack local" (les 3 resultats avec carte Google Maps). Sans fiche GMB, l'entreprise est invisible sur Maps et ne collecte aucun avis Google.

C'est le ticket le plus impactant de tout le backlog.

## Action paysagiste — Etapes

### Etape 1 : Creer la fiche (30 min)
1. Aller sur https://business.google.com
2. Se connecter avec un compte Google (le meme que pour Search Console si possible)
3. Cliquer "Ajouter votre etablissement"
4. Renseigner les infos **NAP strictement identiques au site** :
   - **Nom** : Art des Jardins
   - **Adresse** : 9 bis rue Rouget de l'Isle, 49130 Les Ponts-de-Ce
   - **Telephone** : 07 81 16 07 37
   - **Site web** : https://art-et-jardin.fr
   - **Categorie principale** : Paysagiste
   - **Categories secondaires** : Elagueur, Jardinier, Entreprise d'amenagement paysager

### Etape 2 : Verification (1-2 semaines)
- Google envoie un courrier postal avec un code de verification a l'adresse
- Ou verification par telephone si proposee (plus rapide)
- Entrer le code dans Google My Business pour activer la fiche

### Etape 3 : Completer le profil (1h)
- **Description** : reprendre la description du site (paysagiste a Angers, creation, entretien, elagage...)
- **Horaires** : Lundi-Vendredi 8h-18h
- **Services** : lister les 9 services proposes
- **Photos** : uploader les meilleures realisations (celles du site — au moins 10 photos)
- **Logo** : uploader le logo Art des Jardins

### Etape 4 : Animer le profil (continu)
- Publier 1 post/semaine : photo de chantier en cours, conseil de saison, realisation terminee
- Repondre a chaque avis recu (positif et negatif)

### Etape 5 : Mettre a jour le code (IMP-102)
- Une fois la fiche creee, recuperer l'URL Google Maps du profil
- Me la transmettre pour l'ajouter dans le schema sameAs

## IMPORTANT : Coherence NAP

Le Nom, l'Adresse et le Telephone doivent etre **strictement identiques** partout :
- Site web
- Google My Business
- Pages Jaunes
- Facebook
- Tous les annuaires

La moindre difference (ex: "9B" vs "9 bis", "Les Ponts de Ce" vs "Les Ponts-de-Ce") affaiblit les signaux de citation aupres de Google.
