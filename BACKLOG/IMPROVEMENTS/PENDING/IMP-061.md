# IMP-061: Feedback utilisateur (snackbars, confirmations, haptics)

**Type:** Improvement
**Statut:** A faire
**Priorite:** Haute
**Complexite:** M
**Tags:** ux, feedback
**Date creation:** 2026-02-25

---

## Description

Aucun feedback apres sauvegarde de formulaire. Changement statut chantier sans confirmation (tap accidentel = chantier ferme). Haptic feedback quasi absent (1 seul usage dans AejButton). SignaturePad sans feedback tactile.

## Findings associes

- STATE-010: Pas de feedback de sauvegarde sur formulaires
- STATE-011: Suppression sans undo
- STATE-012: Peu d'utilisation HapticFeedback
- INTER-001: Haptic feedback insuffisant sur actions metier
- INTER-010: SignaturePad sans feedback durant le dessin
- SCREEN-007: Changement statut chantier sans confirmation
- SCREEN-014: Mode terrain sans indication visuelle apres activation

## Criteres d'acceptation

- [ ] Snackbar succes/erreur apres chaque action CRUD
- [ ] Dialog confirmation pour changements de statut chantier
- [ ] Haptic feedback sur actions critiques (signature, suppression, validation)
- [ ] Undo disponible apres suppression (snackbar avec action)
- [ ] Feedback tactile sur SignaturePad (onPanStart/onPanEnd)
