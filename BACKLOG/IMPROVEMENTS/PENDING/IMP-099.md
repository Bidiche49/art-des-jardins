## Description

Cormorant Garamond est charge avec 3 weights (400, 600, 700). Le weight 600 (semibold) est potentiellement inutilise si seuls 400 (normal) et 700 (bold) sont effectivement appliques dans le CSS.

Chaque weight de police Google Font ajoute ~15-30KB de WOFF2 au chargement initial.

## Criteres d'acceptation
- [ ] Verifier l'usage reel de chaque weight dans le CSS/composants
- [ ] Supprimer les weights non utilises
- [ ] Pas de regression visuelle (verifier tous les headings)
- [ ] Font loading plus rapide

## Fichiers concernes
- `apps/vitrine/src/app/layout.tsx` (configuration next/font)
- Rechercher `font-semibold` et `font-serif` dans les composants
