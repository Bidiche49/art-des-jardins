## Description

Telephone, email, adresse, SIRET et autres infos entreprise sont hardcodes dans 20+ fichiers de la vitrine. Si un numero de tel change, il faut modifier manuellement 20+ fichiers. Source d'erreur et cauchemar de maintenance.

`infosvrac.json` existe comme source de verite mais n'est pas importe dans le code.

## Criteres d'acceptation

- [ ] Fichier `apps/vitrine/src/lib/business-info.ts` cree avec toutes les constantes
- [ ] Toutes les pages et composants importent depuis ce fichier
- [ ] Plus aucun tel/email/adresse hardcode dans les pages
- [ ] Build vitrine OK
- [ ] Infos coherentes avec `infosvrac.json`
