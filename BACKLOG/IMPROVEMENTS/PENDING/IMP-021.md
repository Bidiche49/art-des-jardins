## Description

Pas de lien "Aller au contenu principal" pour la navigation clavier. Les utilisateurs clavier doivent tabber a travers tout le header/nav avant d'atteindre le contenu.

## Criteres d'acceptation

- [ ] Lien skip-to-content en premier element du body
- [ ] Visible uniquement au focus clavier (sr-only + focus:not-sr-only)
- [ ] Ancre `#main-content` sur le contenu principal

## Fichiers concernes

- `apps/vitrine/src/app/layout.tsx:90-94`

## Analyse / Approche

```tsx
<body>
  <a href="#main-content" className="sr-only focus:not-sr-only focus:absolute focus:top-4 focus:left-4 focus:z-50 focus:bg-white focus:px-4 focus:py-2 focus:rounded">
    Aller au contenu principal
  </a>
  <Header />
  <main id="main-content">
    {children}
  </main>
</body>
```

## Tests de validation

- [ ] Tab depuis le debut de page → lien visible "Aller au contenu principal"
