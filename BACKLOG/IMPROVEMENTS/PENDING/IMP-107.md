## Description
Google Search Console n'est pas active pour le site. Sans cet outil, impossible de savoir si Google indexe correctement les pages, quelles requetes amenent du trafic, ou s'il y a des erreurs d'exploration. C'est l'equivalent d'un compteur de visites invisible â€” le site existe mais on ne sait pas si Google le voit.

La balise de verification est deja preparee dans le code (commentee) dans `layout.tsx` ligne 97.

## Fichier concerne
- `apps/vitrine/src/app/layout.tsx` (ligne ~97)

## Implementation
Decommenter et remplir le code de verification :
```ts
verification: {
  google: 'VOTRE_CODE_VERIFICATION',
},
```

---

## Ce que le paysagiste doit faire (5 minutes)

### Etape 1 : Creer un compte Google Search Console
1. Aller sur https://search.google.com/search-console
2. Se connecter avec un compte Google (celui qui servira aussi pour Google My Business, idealement)
3. Cliquer "Ajouter une propriete"
4. Choisir "Prefixe de l'URL" et entrer `https://art-et-jardin.fr`

### Etape 2 : Recuperer le code de verification
1. Dans les methodes de verification proposees, choisir **"Balise HTML"**
2. Google affiche une balise du type : `<meta name="google-site-verification" content="abc123XYZ..." />`
3. Copier **uniquement le contenu** (la partie entre guillemets apres `content=`), par exemple : `abc123XYZ...`
4. Me transmettre ce code

### Etape 3 : Je l'integre dans le code
1. J'ajoute le code dans `layout.tsx`
2. Deploiement
3. Retourner dans Search Console et cliquer "Verifier"

### Etape 4 : Soumettre le sitemap
1. Une fois verifie, aller dans Search Console > Sitemaps
2. Entrer `sitemap.xml` et soumettre
3. Google commencera a explorer toutes les pages du site

---

## Pourquoi c'est urgent

Google Search Console est **gratuit** et c'est le seul moyen de :
- Voir si Google indexe bien les 98 pages du site
- Savoir quels mots-cles amenent du trafic (et lesquels non)
- Detecter les erreurs (pages 404, pages lentes, problemes mobile)
- Soumettre le sitemap pour accelerer l'indexation des nouveaux articles
- Mesurer l'impact des ameliorations SEO

**Sans Search Console, tout le travail SEO (blog, pages villes, schema.org) est fait a l'aveugle.**
