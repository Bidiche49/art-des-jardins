# IMP-041: Sticky CTA "Devis gratuit" mobile en bas d'ecran

**Type:** Improvement
**Statut:** Fait
**Date resolution:** 2026-02-16
**Priorite:** Moyenne
**Complexite:** S
**Tags:** vitrine, ui, ux, conversion, mobile
**Date creation:** 2026-02-13

---

## Description

Sur mobile, le CTA principal "Demander un devis gratuit" n'est visible que dans le hero (above the fold) et en fin de page. Quand l'utilisateur scrolle, il n'a plus de CTA visible sauf le bouton flottant Appeler (IMP-031). Un bandeau sticky en bas avec double CTA (Devis + Appeler) maximiserait les conversions.

## User Story

**En tant que** visiteur mobile qui scrolle
**Je veux** pouvoir demander un devis a tout moment
**Afin de** ne pas avoir a remonter en haut de page

## Fichiers concernes

- Nouveau composant : `apps/vitrine/src/components/ui/MobileStickyBar.tsx`
- `apps/vitrine/src/app/layout.tsx` - Inclure le composant

## Approche

```tsx
export function MobileStickyBar() {
  return (
    <div className="fixed bottom-0 inset-x-0 z-40 md:hidden bg-white border-t border-gray-200 shadow-lg p-3">
      <div className="flex gap-3">
        <a
          href="tel:+33781160737"
          className="flex-1 flex items-center justify-center gap-2 py-3 bg-gray-100 text-gray-800 rounded-lg font-medium"
        >
          <PhoneIcon className="w-4 h-4" />
          Appeler
        </a>
        <Link
          href="/contact/"
          className="flex-1 flex items-center justify-center gap-2 py-3 bg-primary-600 text-white rounded-lg font-medium"
        >
          Devis gratuit
        </Link>
      </div>
    </div>
  );
}
```

- **Position** : fixed bottom, pleine largeur
- **Visibilite** : mobile uniquement (md:hidden)
- **Contenu** : 2 boutons cote a cote (Appeler + Devis gratuit)
- **Note** : Si IMP-031 (bouton flottant Appeler) est implemente, ce ticket le remplace. Choisir l'un OU l'autre, pas les deux.
- Ajouter un `padding-bottom` sur le body/main pour eviter que le contenu soit masque par la barre
