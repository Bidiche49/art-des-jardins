# IMP-117: Supprimer scroll-behavior smooth global

**Type:** Improvement
**Statut:** Fait
**Priorité:** Basse
**Complexité:** XS
**Tags:** perf, ux
**Date création:** 2026-02-28

---

## Description

`scroll-behavior: smooth` est appliqué globalement sur `<html>` dans globals.css. Cela force le smooth scroll sur tous les contextes : ancres, retour arrière navigateur, scroll programmatique. Sur mobile, le retour arrière navigateur est visuellement lent.

Le smooth scroll intentionnel (ContactForm.tsx) utilise déjà `window.scrollTo({ behavior: 'smooth' })` qui fonctionne indépendamment du CSS.

## Fix

Supprimer `scroll-behavior: smooth;` dans `globals.css` ligne 7.

- [ ] Ligne `scroll-behavior: smooth` supprimée de globals.css
- [ ] Smooth scroll ContactForm.tsx toujours fonctionnel (utilise window.scrollTo)
- [ ] Navigation par ancre = scroll instantané
- [ ] Retour arrière navigateur = instantané
