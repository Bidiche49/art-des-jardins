## Description

Echanger les photos hero entre les pages Elagage et Abattage pour mieux correspondre au contenu.

### Etat actuel

| Page | Image hero actuelle | Description |
|------|-------------------|-------------|
| Elagage (`/elagage-angers/`) | `elagage-3` | "Resultat de taille d'arbres soignee" |
| Abattage (`/abattage-angers/`) | `elagage-1` | "Grands arbres vus en contre-plongee" |

### Cible

| Page | Image hero cible | Raison |
|------|-----------------|--------|
| Elagage | `elagage-1` | La contre-plongee sur de grands arbres convient mieux a l'elagage |
| Abattage | `elagage-3` | Le resultat de coupe correspond mieux a l'abattage |

### Note

Le paysagiste dit "la photo d'elagage et taille correspond mieux a abattage et inversement". Le swap rend les visuels plus coherents avec le service presente.
