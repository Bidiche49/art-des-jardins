# IMP-046: Harmoniser l'affichage des numeros de telephone sur le site

**Type:** Improvement
**Statut:** Fait
**Priorite:** Haute
**Complexite:** XS
**Tags:** ux, ui, conversion
**Date creation:** 2026-02-16

---

## Description

Actuellement les 2 numeros de telephone apparaissent de facon incoherente sur le site. Harmoniser selon la regle :
- **1 seul numero** dans les zones d'action (header, sticky mobile, CTA)
- **2 numeros** dans les zones d'information (footer, contact, FAQ)

En plus, mentionner WhatsApp dans la FAQ pour offrir un canal de contact supplementaire.

## Numeros

- Principal : **07 81 16 07 37**
- Secondaire : **06 59 68 49 16**
- WhatsApp : via le bouton flottant (meme numero principal)

## Regles par zone

| Zone | Numeros | Etat actuel | Action |
|------|---------|-------------|--------|
| Header desktop | 1 seul (principal) | OK | Rien |
| Sticky bar mobile | 1 seul (principal) | OK | Rien |
| CTA sections | 0 (lien vers /contact) | OK | Rien |
| Footer | 2 numeros | OK | Rien |
| Page Contact | 2 numeros + noms associes | A verifier | Verifier que les 2 y sont avec contexte |
| FAQ (HomeFAQ) | 1 seul | A modifier | Mettre les 2 + mentionner WhatsApp |

## Modification principale : FAQ

Dans `HomeFAQ.tsx`, reponse a "Comment obtenir un devis ?" :

**Avant :**
> Appelez-nous au 07 81 16 07 37 ou remplissez le formulaire...

**Apres :**
> Appelez-nous au 07 81 16 07 37 ou au 06 59 68 49 16, ecrivez-nous sur WhatsApp, ou remplissez le formulaire...
