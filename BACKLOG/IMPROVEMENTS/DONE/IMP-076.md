## Description

Ajouter un lien vers le compte Instagram d'Art des Jardins a plusieurs endroits du site.

**Compte Instagram** : `https://www.instagram.com/artdesjardins_49`

### Emplacements a ajouter

1. **Footer** : Icone Instagram dans la section contact/liens (a cote du tel/email)
2. **Section realisations** (homepage) : Bouton/lien "Voir plus sur Instagram" sous la galerie
3. **Schema.org** : Ajouter l'URL Instagram dans `sameAs` du `LocalBusinessSchema`

### Note SEO

- L'ajout du lien Instagram dans le schema.org `sameAs` renforce le Knowledge Graph Google
- Le lien social est un signal de confiance pour le SEO local
- Attribut `rel="noopener noreferrer"` + `target="_blank"` pour les liens externes

## Fichiers concernes

- `apps/vitrine/src/components/layout/Footer.tsx` (icone + lien)
- `apps/vitrine/src/app/page.tsx` (section realisations, lien "Voir sur Instagram")
- `apps/vitrine/src/components/seo/LocalBusinessSchema.tsx` (sameAs)
