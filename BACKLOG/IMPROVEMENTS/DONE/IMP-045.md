# IMP-045: Egaliser la hauteur des ServiceCard dans la grille accueil

**Type:** Improvement
**Statut:** Fait
**Priorite:** Basse
**Complexite:** XS
**Tags:** ui, ux
**Date creation:** 2026-02-16

---

## Description

Sur la page d'accueil, section "Nos services de paysagiste à Angers", les 4 `ServiceCard` sont affichées en grille (`grid md:grid-cols-2 lg:grid-cols-4`). Quand un titre comme "Aménagement paysager" passe sur 2 lignes en responsive (breakpoints intermédiaires), les cartes n'ont pas toutes la même hauteur. Les 4 cellules doivent rester alignées quelle que soit la longueur du titre.

## User Story

**En tant que** visiteur du site
**Je veux** que les 4 cartes de services soient visuellement alignées
**Afin de** avoir un rendu propre et professionnel à tous les breakpoints

## Criteres d'acceptation

- [x] Les 4 ServiceCard ont la même hauteur sur une même ligne de grille
- [x] Le titre peut passer sur 2 lignes sans casser l'alignement
- [x] La description et le lien "En savoir plus" restent alignés entre les cartes
- [x] Mobile (1 col, layout horizontal image/texte), tablette (2 cols), desktop (4 cols)

## Fichiers concernes

- `apps/vitrine/src/components/ui/ServiceCard.tsx`
- `apps/vitrine/src/app/page.tsx` (grille parente)

## Analyse / Approche

La grille CSS `grid` égalise déjà la hauteur des cellules par ligne. Le problème vient probablement de l'intérieur de la `ServiceCard` : le titre, la description et le CTA ne sont pas distribués avec `flex-col` + `flex-grow` pour que les zones s'alignent entre cartes.

**Fix probable :** Ajouter `h-full flex flex-col` sur le conteneur de la ServiceCard, et `flex-grow` sur la zone description pour pousser le CTA "En savoir plus" tout en bas de chaque carte.
