## Description

Le paysagiste souhaite supprimer les legendes visibles sous les photos de la galerie de realisations sur la page d'accueil.

### Etat actuel

Le composant `PhotoGallery` affiche des photos avec des legendes (categorie, description) sous chaque image.

### Cible

- Supprimer les legendes/textes visibles sous les photos de la galerie
- **Conserver les alt text** (invisible, essentiel pour SEO et accessibilite)
- Les filtres par categorie restent si presents

### Note UX/SEO

Les alt text restent en place pour le referencement images Google. Seules les legendes visibles sont supprimees. C'est une bonne decision UX : les photos parlent d'elles-memes.

## Fichiers concernes

- `apps/vitrine/src/components/ui/PhotoGallery.tsx`

## Criteres d'acceptation

- [ ] Aucune legende visible sous les photos de la galerie
- [ ] Alt text toujours presents sur les balises img
- [ ] Layout de la galerie reste coherent sans legendes
- [ ] Pas de regression sur les autres pages utilisant PhotoGallery
