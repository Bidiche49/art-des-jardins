# IMP-039: Ajouter Organization schema dedie et enrichir LocalBusiness

**Type:** Improvement
**Statut:** Fait
**Priorite:** Moyenne
**Complexite:** S
**Tags:** seo, vitrine, schema
**Date creation:** 2026-02-13

---

## Description

Le site utilise uniquement `LocalBusiness` comme type de schema. Pour une couverture optimale, il faut aussi un schema `Organization` dedie et enrichir le `LocalBusiness` avec des champs manquants : `logo`, `foundingDate`, `numberOfEmployees`, `knowsLanguage`.

De plus, le `LocalBusinessCitySchema` (utilise sur les pages ville) ne reprend pas les coordonnees GPS ni l'adresse complete de l'entreprise.

## Fichiers concernes

- `apps/vitrine/src/components/seo/LocalBusinessSchema.tsx`
- `apps/vitrine/src/components/seo/LocalBusinessCitySchema.tsx`

## Approche

### 1. Enrichir LocalBusinessSchema
```typescript
logo: 'https://art-et-jardin.fr/images/logo.png',
foundingDate: '2024', // ou date reelle
numberOfEmployees: {
  '@type': 'QuantitativeValue',
  value: 2,
},
knowsLanguage: 'French',
paymentAccepted: ['Cash', 'Check', 'Bank Transfer'],
currenciesAccepted: 'EUR',
```

### 2. Enrichir LocalBusinessCitySchema
Ajouter les geo coordinates et l'adresse complete du siege (meme pour les pages ville, l'adresse reste celle du siege) :
```typescript
geo: {
  '@type': 'GeoCoordinates',
  latitude: 47.4264,
  longitude: -0.5256,
},
```

### 3. Optionnel - Organization schema
Ajouter un schema `Organization` en complement dans le layout :
```typescript
{
  '@context': 'https://schema.org',
  '@type': 'Organization',
  name: 'Art des Jardins',
  url: 'https://art-et-jardin.fr',
  logo: 'https://art-et-jardin.fr/images/logo.png',
  contactPoint: { ... },
  sameAs: [ ... ],
}
```
