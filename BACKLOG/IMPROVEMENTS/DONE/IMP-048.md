# IMP-048: Responsive mobile footer

**Type:** Improvement
**Statut:** Fait
**Priorite:** Basse
**Complexite:** XS
**Tags:** ui, ux
**Date creation:** 2026-02-16

---

## Description

Le footer de la vitrine n'etait pas optimise pour mobile : les colonnes Services et Informations s'empilaient verticalement en prenant trop de place, les numeros de telephone risquaient de deborder, et les zones tactiles des liens etaient trop petites.

## User Story

**En tant que** visiteur mobile du site
**Je veux** un footer compact et ergonomique
**Afin de** naviguer facilement vers les liens utiles sur petit ecran

## Criteres d'acceptation

- [x] Services et Informations cote a cote en 2 colonnes sur mobile
- [x] Les 2 numeros de telephone restent sur la meme ligne a tous les breakpoints
- [x] Zones tactiles des liens agrandies sur mobile (py-1.5)
- [x] Layout 4 colonnes preserve sur tablette et desktop
- [x] Technique md:contents pour eviter la duplication de markup

## Fichiers concernes

- `apps/vitrine/src/components/layout/Footer.tsx`

## Tests de validation

- [x] Viewport 375px : 2 colonnes liens, numeros sur 1 ligne, liens cliquables facilement
- [x] Viewport 768px+ : 4 colonnes classiques, espacement normal
