# IMP-033: Ameliorer alt text descriptifs sur toutes les images

**Type:** Improvement
**Statut:** Fait
**Priorite:** Moyenne
**Complexite:** S
**Tags:** seo, vitrine, accessibilite
**Date creation:** 2026-02-13

---

## Description

Les alt text des images sont generiques (`alt={service.title}` -> "Amenagement Paysager") au lieu d'etre descriptifs du contenu reel de l'image. Des alt text optimises ameliorent a la fois le SEO images (Google Images) et l'accessibilite.

## User Story

**En tant que** moteur de recherche / lecteur d'ecran
**Je veux** comprendre ce que represente chaque image
**Afin de** indexer correctement l'image / decrire l'image aux malvoyants

## Fichiers concernes

- `apps/vitrine/src/app/services/page.tsx` - Images services
- `apps/vitrine/src/app/page.tsx` - Images hero et services
- `apps/vitrine/src/components/ui/ServiceCard.tsx` - Alt generique
- `apps/vitrine/src/components/ui/PhotoGallery.tsx` - Alt des photos
- `apps/vitrine/src/components/ui/InlineGallery.tsx` - Alt des photos
- `apps/vitrine/src/lib/images-manifest.ts` - Ajouter les alt descriptifs

## Approche

1. **Ajouter un champ `altText` dans le manifest d'images** :
```typescript
// images-manifest.ts
export const imageAlts: Record<string, string> = {
  'creation-1': 'Jardin contemporain avec terrasse bois et massifs fleuris a Angers',
  'creation-2': 'Amenagement paysager complet avec pelouse et plantation d\'arbustes',
  'elagage-1': 'Elagueur professionnel en nacelle taillant un chene centenaire',
  'elagage-2': 'Taille d\'entretien d\'un tilleul dans un jardin a Angers',
  'entretien-1': 'Tonte de pelouse et entretien de massifs par un jardinier professionnel',
  // ... etc
};
```

2. **Utiliser ces alt dans tous les composants** au lieu de `alt={service.title}`

3. **Format optimal pour les alt SEO** : `[description de l'image] a [lieu]`
   - Bon : "Creation d'un jardin avec terrasse bois et massifs fleuris a Angers"
   - Mauvais : "Amenagement Paysager" (trop generique)

## Criteres d'acceptation

- [ ] Chaque image a un alt text unique et descriptif
- [ ] Les alt contiennent des mots-cles locaux quand pertinent
- [ ] Les alt decrivent reellement le contenu de l'image
- [ ] Les alt ne font pas de keyword stuffing
- [ ] Max 125 caracteres par alt text

## Tests de validation

- [ ] Inspecter le HTML et verifier chaque attribut alt
- [ ] Test avec un lecteur d'ecran (VoiceOver)
- [ ] Desactiver les images dans le navigateur et verifier que les alt sont coherents
