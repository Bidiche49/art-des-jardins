## Description

Le site ne mentionne aucune certification, assurance ou numero SIRET de maniere visible. Pour un artisan paysagiste, ces elements sont **essentiels pour la confiance** :
- Assurance decennale / responsabilite civile pro
- Certifications (Qualipaysage, CS Elagage, etc.)
- Numero SIRET
- Annees d'experience clairement affichees

Sans ces elements, les visiteurs peuvent douter du serieux de l'entreprise.

## User Story

**En tant que** particulier cherchant un paysagiste
**Je veux** voir les preuves de professionnalisme
**Afin de** etre rassure avant de demander un devis

## Fichiers concernes

- `apps/vitrine/src/app/page.tsx` - Section confiance a ajouter
- `apps/vitrine/src/components/layout/Footer.tsx` - Ajouter SIRET
- `apps/vitrine/src/components/seo/LocalBusinessSchema.tsx` - Enrichir schema

## Approche

### 1. Bandeau de confiance sur la homepage (sous le hero ou sous les services)
```tsx
<section className="py-8 bg-white border-y border-gray-100">
  <div className="container-custom">
    <div className="flex flex-wrap justify-center items-center gap-8 text-gray-600 text-sm">
      <div className="flex items-center gap-2">
        <ShieldIcon /> Assurance RC Pro
      </div>
      <div className="flex items-center gap-2">
        <AwardIcon /> 16 ans d'experience cumulee
      </div>
      <div className="flex items-center gap-2">
        <CheckIcon /> Devis gratuit sous 48h
      </div>
      <div className="flex items-center gap-2">
        <StarIcon /> 4.8/5 satisfaction client
      </div>
    </div>
  </div>
</section>
```

### 2. SIRET dans le footer
Ajouter sous le copyright :
```
SARL Art des Jardins - SIRET XXX XXX XXX XXXXX - RCS Angers
```

### 3. Schema enrichi
Ajouter dans LocalBusinessSchema :
```typescript
identifier: {
  '@type': 'PropertyValue',
  name: 'SIRET',
  value: 'XXX XXX XXX XXXXX',
},
```

**Pre-requis** : Recuperer aupres du client le SIRET exact, le nom des assurances, les certifications.
