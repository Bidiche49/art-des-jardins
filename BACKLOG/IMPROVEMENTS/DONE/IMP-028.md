# IMP-028: Ajouter telephone cliquable dans le header desktop

**Type:** Improvement
**Statut:** Fait
**Date resolution:** 2026-02-16
**Priorite:** Haute
**Complexite:** XS
**Tags:** vitrine, ui, ux, conversion, seo
**Date creation:** 2026-02-13

---

## Description

Le header desktop n'affiche pas de numero de telephone. Les visiteurs doivent naviguer jusqu'a la page contact ou le footer pour trouver le numero. Pour un site d'artisan paysagiste, le telephone est le **canal de conversion principal** - il doit etre visible immediatement.

## User Story

**En tant que** visiteur sur desktop
**Je veux** voir le numero de telephone dans le header
**Afin de** pouvoir appeler directement sans chercher

## Fichiers concernes

- `apps/vitrine/src/components/layout/Header.tsx`

## Approche

Ajouter a gauche du bouton "Devis gratuit" dans la navigation desktop :
```tsx
<a href="tel:+33781160737" className="flex items-center gap-2 text-gray-700 hover:text-primary-600">
  <PhoneIcon className="w-4 h-4" />
  <span className="font-medium">07 81 16 07 37</span>
</a>
```

- Position : entre les liens de navigation et le bouton CTA
- Style : discret mais visible, avec icone telephone
- Ne PAS afficher sur mobile (le bouton flottant s'en chargera - voir IMP-031)
- Classe `hidden lg:flex` pour masquer en dessous de lg

## Criteres d'acceptation

- [ ] Numero de telephone visible dans le header desktop (lg+)
- [ ] Lien `tel:` cliquable
- [ ] Icone telephone a cote du numero
- [ ] Masque sur mobile et tablette
- [ ] Style coherent avec le reste du header
- [ ] Le header ne deborde pas avec le numero ajoute

## Tests de validation

- [ ] Verifier visuellement sur desktop (1024px+)
- [ ] Verifier que le numero est masque en mobile
