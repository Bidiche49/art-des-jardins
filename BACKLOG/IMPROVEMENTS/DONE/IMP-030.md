## Description

La page contact affiche un placeholder gris a la place d'une carte Google Maps. Une vraie carte avec le marqueur de l'entreprise renforce la confiance, aide au SEO local et permet aux clients de visualiser la zone d'intervention.

## User Story

**En tant que** visiteur potentiel
**Je veux** voir une carte avec la localisation de l'entreprise
**Afin de** verifier que l'entreprise intervient dans ma zone

## Fichiers concernes

- `apps/vitrine/src/app/contact/page.tsx` (lignes 222-252)

## Approche

**Option A - iframe Google Maps (recommande, gratuit, simple) :**
```tsx
<iframe
  src="https://www.google.com/maps/embed?pb=!1m18!..."
  width="100%"
  height="320"
  style={{ border: 0 }}
  allowFullScreen
  loading="lazy"
  referrerPolicy="no-referrer-when-downgrade"
  title="Art des Jardins - Les Ponts-de-Ce"
  className="rounded-xl"
/>
```

1. Aller sur Google Maps
2. Rechercher "9 bis rue Rouget de l'Isle, 49130 Les Ponts-de-Ce"
3. Cliquer "Partager" > "Integrer une carte"
4. Copier le code iframe
5. Remplacer le placeholder dans contact/page.tsx

**Option B - Carte statique (si probleme performance) :**
Image statique Google Maps avec lien vers Google Maps en pleine page.

**Ajouter aussi `hasMap` dans le schema :**
```typescript
// Dans LocalBusinessSchema.tsx
hasMap: 'https://www.google.com/maps/place/...',
```
