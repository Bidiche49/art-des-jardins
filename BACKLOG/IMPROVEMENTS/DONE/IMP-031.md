## Description

Sur mobile, le numero de telephone n'est accessible qu'en scrollant jusqu'au footer ou en allant sur la page contact. Pour un site d'artisan, **80%+ du trafic est mobile** et le telephone est le premier canal de conversion. Un bouton flottant "Appeler" toujours visible est indispensable.

## User Story

**En tant que** visiteur mobile
**Je veux** pouvoir appeler l'entreprise en un clic a tout moment
**Afin de** demander un devis sans chercher le numero

## Fichiers concernes

- `apps/vitrine/src/app/layout.tsx` (ajouter le composant)
- Nouveau composant : `apps/vitrine/src/components/ui/FloatingCallButton.tsx`

## Approche

Creer un composant `FloatingCallButton` :
```tsx
export function FloatingCallButton() {
  return (
    <a
      href="tel:+33781160737"
      className="fixed bottom-6 right-6 z-40 md:hidden flex items-center gap-2 bg-primary-600 text-white rounded-full px-5 py-3 shadow-lg hover:bg-primary-700 transition-colors"
      aria-label="Appeler Art des Jardins"
    >
      <PhoneIcon className="w-5 h-5" />
      <span className="font-medium text-sm">Appeler</span>
    </a>
  );
}
```

Caracteristiques :
- **Position** : fixed, bottom-right, z-40
- **Visibilite** : mobile uniquement (`md:hidden`)
- **Style** : pilule verte avec icone telephone + texte "Appeler"
- **Ombre** : shadow-lg pour se detacher du contenu
- **Animation** : optionnellement un leger pulse pour attirer l'attention
- **Accessibilite** : aria-label descriptif
- Ne doit pas chevaucher les autres elements (footer, etc.)
- Disparait au scroll vers le footer (optionnel)
