# IMP-029: Remplir sameAs dans LocalBusinessSchema

**Type:** Improvement
**Statut:** Fait
**Priorite:** Haute
**Complexite:** XS
**Tags:** seo, vitrine, schema
**Date creation:** 2026-02-13

---

## Description

Le champ `sameAs` dans le schema LocalBusiness est un tableau vide `[]`. Ce champ doit contenir les URLs des profils de l'entreprise sur les plateformes externes pour renforcer l'E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness) aux yeux de Google.

## User Story

**En tant que** moteur de recherche
**Je veux** connaitre les profils officiels de l'entreprise
**Afin de** verifier sa legitimite et enrichir le Knowledge Panel

## Fichiers concernes

- `apps/vitrine/src/components/seo/LocalBusinessSchema.tsx` (ligne 56)

## Approche

Remplir `sameAs` avec les profils existants ou a creer :
```typescript
sameAs: [
  'https://www.google.com/maps/place/Art+des+Jardins/...', // Fiche Google Business Profile
  'https://www.pagesjaunes.fr/pros/...', // Pages Jaunes
  'https://www.facebook.com/artdesjardins49/', // Facebook (si existe)
  'https://www.instagram.com/artdesjardins49/', // Instagram (si existe)
  'https://www.linkedin.com/company/art-des-jardins/', // LinkedIn (si existe)
],
```

**Pre-requis** : Verifier quels profils existent deja. Si certains n'existent pas, les creer d'abord (hors scope de ce ticket).

A minima, **Google Business Profile** et **Pages Jaunes** doivent etre renseignes car ce sont les plus importants pour le SEO local en France.

## Criteres d'acceptation

- [ ] `sameAs` contient au minimum le lien Google Business Profile
- [ ] `sameAs` contient le lien Pages Jaunes si la fiche existe
- [ ] Les URLs sont valides et pointent vers les bons profils
- [ ] Le schema JSON-LD genere est valide (tester sur schema.org validator)
