## Description

Le site vitrine utilisait un vert vif "tech startup" (`#16a34a`, contraste 3.30:1) et la police Inter partout. Le logo Art des Jardins a des tons olive/foret naturels et une police serif elegante. Alignement de l'identite visuelle du site sur le logo.

## User Story

**En tant que** visiteur du site
**Je veux** une identite visuelle coherente entre le logo et le site
**Afin de** percevoir une marque professionnelle et naturelle (paysagiste)

## Criteres d'acceptation

- [x] Palette primaire olive/foret (extraite du logo) remplace le vert vif
- [x] Palette secondaire or/bronze (branches du logo) remplace le jaune
- [x] Police Cormorant Garamond sur h1, h2 et texte logo
- [x] Hero overlays ajustes (teinte olive chaude)

- [x] Build production passe (98 pages)

## Analyse / Approche

### Palette primaire (olive/foret)

| Shade | Hex | Usage |
|-------|-----|-------|
| 50 | #f6f7f0 | Fonds legers |
| 100 | #e9eedd | Conteneurs icones |
| 200 | #d3debb | Gradients |
| 300 | #b6c78f | Bordures deco |
| 400 | #96ac66 | Deco non-texte |
| 500 | #738847 | Focus rings (3.93:1) |
| 600 | #576b3b | Boutons, liens (5.88:1 AA) |
| 700 | #44542f | Hover, texte sombre (8.22:1 AAA) |
| 800 | #384429 | Titres forts (10.36:1) |
| 900 | #303b24 | Overlays sombres (11.82:1) |

### Police serif

Cormorant Garamond (Google Fonts) - plus proche du style logo que Playfair Display.
Appliquee en `@layer base` sur h1/h2 et en `font-serif` explicite sur le texte logo.
