# IMP-043: Ajouter icones SVG accessibles avec aria-label dans le footer

**Type:** Improvement
**Statut:** Fait
**Priorite:** Basse
**Complexite:** XS
**Tags:** vitrine, accessibilite
**Date creation:** 2026-02-13

---

## Description

Les icones SVG dans le footer (localisation, telephone, email) n'ont pas de `role="img"` ni de `aria-label`. Les lecteurs d'ecran ne peuvent pas decrire ces icones aux utilisateurs malvoyants.

## Fichiers concernes

- `apps/vitrine/src/components/layout/Footer.tsx`

## Approche

Ajouter `role="img"` et `aria-hidden="true"` sur les SVG decoratives (celles accompagnees de texte) :

```tsx
<svg className="w-4 h-4 text-primary-500 flex-shrink-0" fill="currentColor" viewBox="0 0 20 20" aria-hidden="true">
```

Puisque toutes les icones du footer sont accompagnees de texte (adresse, telephone, email), `aria-hidden="true"` est la bonne approche - l'icone est decorative et le texte a cote suffit.
