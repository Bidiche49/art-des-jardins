## Description
6+ formulations differentes de CTA sur le site creaient de la confusion cognitive. Unifie en 2 formulations :
- Hero homepage : "Demander ma visite gratuite" (differenciant)
- Partout ailleurs : "Demander un devis gratuit" (convertit mieux en France pour artisan)
- Header : "Devis gratuit" (court, attendu)

"Devis" reste le mot qui convertit le mieux dans le secteur artisan FR. "Visite" est reserve au hero homepage comme hook differenciant.
