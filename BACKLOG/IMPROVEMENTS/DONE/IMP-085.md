# IMP-085: Ajouter CTA entre sections Before/After et FAQ

**Type:** Improvement
**Statut:** A faire
**Priorite:** Haute
**Complexite:** XS
**Tags:** vitrine, cro, conversion
**Date creation:** 2026-02-27

## Description

Sur la homepage, apres la section Before/After (sliders interactifs montrant le travail realise), il n'y a aucun CTA. L'utilisateur est dans un etat emotionnel positif (impressionne par les transformations) mais n'a aucune invitation a passer a l'action. Il tombe directement dans la FAQ.

Un CTA intermediaire capitaliserait sur cette emotion pour convertir.

## User Story
**En tant que** visiteur impressionne par les avant/apres
**Je veux** pouvoir demander un devis immediatement
**Afin de** ne pas perdre ma motivation

## Criteres d'acceptation
- [ ] CTA visible entre BeforeAfterSection et HomeFAQ
- [ ] Texte type "Votre jardin merite aussi cette transformation"
- [ ] Bouton "Demander un devis gratuit" vers /contact
- [ ] Style coherent avec le CTA final de la page

## Fichiers concernes
- `apps/vitrine/src/app/page.tsx`

## Analyse / Approche
Ajouter une section CTA simple (titre + sous-titre + bouton) entre `<BeforeAfterSection />` et `<HomeFAQ />` dans page.tsx.
