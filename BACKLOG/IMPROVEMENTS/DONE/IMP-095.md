**Statut:** A faire

**Tags:** vitrine, seo, schema

## Description

Les pages de detail service ont une section "Comment ca marche ?" avec 4-5 etapes. Ce contenu peut etre balisage avec le schema `HowTo` de schema.org pour obtenir des rich snippets dans Google (etapes affichees directement dans les resultats).

### Services concernes
- Amenagement paysager (5 etapes : visite, conception, validation, realisation, suivi)
- Entretien de jardin (4 etapes)
- Elagage (4 etapes)
- Abattage (5 etapes)

## Criteres d'acceptation
- [ ] Schema HowTo JSON-LD sur chaque page service detail
- [ ] Chaque etape a un name et text
- [ ] Schema valide (test avec Google Rich Results)
- [ ] Pas de conflit avec les schemas existants (Service, FAQ)

## Fichiers concernes
- `apps/vitrine/src/app/services/[slug]/page.tsx`
- `apps/vitrine/src/lib/services-data.ts` (donnees des etapes)
