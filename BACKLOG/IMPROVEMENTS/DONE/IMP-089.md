# IMP-089: Completer sameAs dans schema LocalBusiness

**Type:** Improvement
**Statut:** A faire
**Priorite:** Moyenne
**Complexite:** XS
**Tags:** vitrine, seo, schema
**Date creation:** 2026-02-27

---

## Description

Le schema LocalBusiness n'a qu'un seul lien `sameAs` (Instagram). Les autres reseaux/annuaires sont commentes dans le code. Pour maximiser les signaux d'autorite SEO, il faut completer avec toutes les presences en ligne de l'entreprise.

### sameAs a ajouter (si les profils existent)
- Google Business Profile URL
- PagesJaunes / Solocal
- Facebook (si page pro existe)
- LinkedIn (si page entreprise)
- Kompass, Societe.com, etc.

### Action requise
Demander au paysagiste quels profils en ligne existent et recuperer les URLs.

## Criteres d'acceptation
- [ ] Au minimum Google Business Profile ajoute (si existant)
- [ ] Tous les profils existants listes dans sameAs
- [ ] Pas de liens vers des profils inexistants

## Fichiers concernes
- `apps/vitrine/src/components/LocalBusinessSchema.tsx` (ou equivalent)
