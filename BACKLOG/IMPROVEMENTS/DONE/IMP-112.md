# IMP-112: Ameliorer visibilite CTA hero homepage

**Type:** Improvement
**Statut:** Fait
**Priorite:** Haute
**Complexite:** XS
**Tags:** cro, ui, homepage
**Date creation:** 2026-02-28

## Description
Le CTA principal "Demander ma visite gratuite" etait en btn-primary-light (blanc) sur un hero avec texte blanc et overlay clair. Il se fondait visuellement avec le CTA secondaire, sans hierarchie visuelle.

## Implementation
Passe le CTA principal de `btn-primary-light` a `btn-primary` (fond vert gradient) pour creer un point focal immediat. Le secondaire reste transparent/blanc.

- `apps/vitrine/src/app/page.tsx`
